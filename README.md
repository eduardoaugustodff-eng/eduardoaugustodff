# AUTO PEÇAS NUNEDIESEL - Catálogo Cummins

Catálogo interativo de peças e especificações para Motor Cummins SÉRIE C 8.3L

## 📋 Acessar o Catálogo

### Online (GitHub Pages)
```
https://eduardoaugustodff-eng.github.io/eduardoaugustodff/catalogo-cummins.html
```

### Localmente
1. Clone o repositório
2. Abra `catalogo-cummins.html` no navegador

## 🎯 Funcionalidades

- 🔍 **Busca avançada** por código, descrição ou número de peça
- 📦 **14 categorias** de componentes Cummins
- 🎴 **Visualização dupla**: Cards e Tabela
- 📱 **Responsivo** para todos os dispositivos
- ⚡ **Design moderno** com tema escuro profissional

## 📂 Arquivos

- `catalogo-cummins.html` - Catálogo interativo
- `cummins-specs.json` - Dados estruturados das peças
