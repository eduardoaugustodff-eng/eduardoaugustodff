Catálogos interativos de peças e especificações para Motores Cummins Série B (6BT) e Série C (8.3L)

## 📁 Versões Disponíveis

### Motor Cummins 6BT (Série B)
- **Deslocamento**: 5.9L (360 ci)
- - **Potência**: 160-220 HP
  - - **Torque**: 520-600 Nm
    - - **Especificações completas**: https://eduardoaugustodff-eng.github.io/eduardoaugustodff/refcumminsb.html
      - - **Arquivo Local**: `refcumminsb.html`

          - - **Especificações completas**: https://eduardoaugustodff-eng.github.io/eduardoaugustodff/catalogo-cummins.html
